# Static Files

Use this skill when exposing static assets via Litestar.

## Workflow

1. Mount static file handling at explicit paths.
2. Set cache headers for immutable assets.
3. Keep static root boundaries locked down.
4. Use reverse proxy/CDN strategy for production scale.
