# OpenAPI

Use this skill when generating or refining API documentation/schema.

## Workflow

1. Configure app-level OpenAPI metadata and docs endpoints.
2. Add operation-level metadata on handlers.
3. Tune schema info for models and edge-case types.
4. Validate generated docs against real response/request behavior.

## Coverage

- Basic OpenAPI setup.
- Route operation customization.
- Schema information customization.
