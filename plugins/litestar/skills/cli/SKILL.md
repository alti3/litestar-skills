# CLI

Use this skill when configuring `litestar` command-line workflows.

## Workflow

1. Define app import path and environment configuration.
2. Use CLI commands for local run/dev tasks.
3. Add project scripts for repeatable team usage.
4. Keep production startup command explicit and environment-safe.

## Checklist

- Ensure app discovery path is deterministic.
- Separate dev and production command presets.
- Document required env vars in project config.
